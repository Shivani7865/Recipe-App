# 🍲 Recipe App

A simple recipe search web app using TheMealDB API.

## 🚀 Features
- Search recipes by name
- View ingredients and instructions
- Clean UI with animations

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- API: TheMealDB

- index.html
- style.css
- script.js
